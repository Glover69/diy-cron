# DIY Cron

The frontend for DIY Cron, a self-hosted cron job scheduling platform. Built with Angular.

> ⚠️ This project is currently in active development and not yet open for public use.

---

## What is DIY Cron?

DIY Cron is a platform that lets developers schedule HTTP jobs on a cron schedule, chain them together, monitor execution history, and get notified via webhooks when jobs succeed or fail.
